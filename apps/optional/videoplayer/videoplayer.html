Video Player Optional App

Play USB-hosted videos directly in your browser. Paste a relative path from File Manager or open this app from a file link.
No video loaded. Enter a path and click Play.
Tips:
• Use paths relative to your USB root (same as File Manager).
• Supports: mp4, mkv, mov, avi, webm, m4v.
• If opened from File Manager, append ?path=your/video.mp4 to this app URL.
